Description:

This is an automation for Nginx and VMA workloads.

-----------------------------------------------------------------------------------------------------------------------

Structure:
run
├── README.txt           - This file.
├── configuration.json   - Configuration file for the automation.
├── __init__.py
├── common.py            - Common utility function used in different scripts.
├── run_automation.py    - The main script of the automation, uses all other scripts in order to operate.
├── run_server.py        - Responsible to run everything related to the server.
├── run_client.py        - Responsible to run everything related to the client.
└── parse_results.py     - The parser of the results the automation creates, generate CSV files with the results.

-----------------------------------------------------------------------------------------------------------------------

Prerequisites:
    1. Standard Linux machine.
    2. The automation uses SSH to run on the remote machines, thus it is required to have free
       access to the machines used.
       Copy SSH keys to all the players used in the automation, server and client machines:
       Command: ssh-copy-id -i <SSH_PUBLIC_KEY> -f <HOST_NAME>
       There are sometime issues with this, search for solutions online.

-----------------------------------------------------------------------------------------------------------------------

Run:
    1. Configure the automation using configuration.json file.
    2. Run the automation(from a machine that is not one of the players): ./run_automation.py
    3. Check out the results in the folder you configured.
    4. Optional step: you can use the Excel template file in the repo.
       Copy the tables from the CSV files generated by the automation to the template.
